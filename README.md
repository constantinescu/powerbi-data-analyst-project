# 🏦 Bank Loan Analysis – Power BI Project

## 📌 Overview
This project was developed as part of my Data Analyst course.  
It uses Power BI to analyze a bank’s loan portfolio and provide insights into loan distribution, customer segments, profitability, and geographic performance.

The goal of this analysis is to help understand lending behavior and support data-driven business decisions such as branch expansion, product optimization, and risk management.

---

## 🎯 Business Questions
- Which branch (city) has the highest loan volume?
- What type of loan is most commonly granted? (Mortgage, Consumer, Auto, Business)
- Which customer segment (Retail, SME, Corporate) takes the largest loans?
- In which cities are large loans granted, and in which cities are small loans granted?
- Which loan types have the highest interest rates?
- Which customer segment is the most profitable for the bank?
- Are large loans issued for short or long periods?
- Is there an increase in loans over time?
- Is the bank’s portfolio concentrated in a single city or distributed across multiple cities?
- If you were a manager, in which city would you open a new branch?

### 🏦 Loan Distribution & Volume
- Which branch (city) has the highest loan volume?
- Are loans concentrated in a few cities or distributed across regions?
- In which cities are large loans granted, and where are smaller loans more common?

### 💳 Loan Products
- What type of loan is most commonly granted? (Mortgage, Consumer, Auto, Business)
- Which loan types have the highest interest rates?

### 👥 Customer Segments
- Which customer segment (Retail, SME, Corporate) takes the largest loans?
- Which segment is the most profitable for the bank?
